ES6-变量的解构赋值

发表于 2018-03-15 | 分类于 ES6

###解构赋值

ES6允许按照一定模式，从数组和对象那个中提取值，对变量进行赋值，这被称为解构。解构赋值在实际开发中可以大量减少我们的代码了，并且让我们的程序解构更清晰。

let [a,b,c]=[1,2,3];

上面的代码表示，可以从数组中提取值，按照位置的对象关系对变量赋值。

###解构的默认值

解构赋值是允许使用默认值的

let [a,b='liuyang'] = ['web']
console.log(a+b); // 'webliuyang'

需要注意下undefined和null的区别，undefined相当于什么都没有，null有值，但为空。

###对象的解构赋值

let {foot,bar} = { foo:'liuyang',bar:'gggg'};
console.log(a+b); // 'liuyanggggg'

注意：对象的解构与数组有一个重要的不同。数组的元素是按次序排列的，变量的取值由它的位置决定；而对象的属性没有次序，变量必须与属性同名，才能取到正确的值。

###字符串解构

const [a,b,c,d,e,f]="liuyang";
