NVS node版本切换工具

发表于 2018-05-30 | 分类于 node版本切换工具

####在项目中node的版本可能和现在电脑上的node版本不一致，导致项目无法运行，我在ionic的项目就碰到这种情况，我的项目是node 6.x版本的，但是电脑是node 8.x版本的。网上的解决方法是安装node的n模块，来切换版本，但是我电脑是windows系统，无法安装使用。

####后来找到了NVS版本切换工具，成功切换版本，运行项目。

下载地址：https://github.com/jasongin/nvs/releases

github：https://github.com/jasongin/nvs

里面有使用文档

#####常用命令：

添加最新版本的节点或指定版本

nvs add latest 或   nvs add [version]

查看本地节点版本 

nvs ls

在当前sell中使用节点版本

nvs use [version]
